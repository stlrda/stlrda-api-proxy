Regional Data Alliance API Server

Welcome to the Regional Data API. This server allows programmatic access to the datasets that we make available. Further documentation is available for each of the following projects. You may click a link to be automatically redirected to the correct documentation.


⚠️ Please note: These API's are currently in a "frozen" state, meaning they will continue to serve data as they have before, but new data will not be made available. This is due to a number of changes in public data sharing as well as a shift in the technical priorities of the RDA.


If you have any questions or concerns, please email bransonfox@umsl.edu
